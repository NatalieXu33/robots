boolean com.mysql.jdbc.LoadBalanceExceptionChecker.shouldExceptionTriggerFailover ( SQLException  ex)

Invoked to determine whether or a given SQLException should trigger a failover in a load-balanced deployment.

The driver will not pass in a Connection instance when calling init(), but it will pass in the Properties, otherwise it acts like a normal Extension.

One instance of a handler per JDBC connection instance will be created. If you need singleton-like behavior, you're on your own to provide it.

true if the exception should trigger failover.
